# JS Utils

Flywheel JS utils is a library of useful JavaScript utilities.

## Install
You can install it via NPM:

```shell
$ npm -i fwsp-jsutils
```

## Usage

```javascript
const Utils = require('fwsp-jsutils');
let hash = Utils.stringHash('This is a test');
```

## Tests

Tests can be found in the `specs` folder.

```shell
$ npm test
```
